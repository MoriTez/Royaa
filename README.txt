
آموزش نصب و اجرای ربات تحلیل چهره تلگرام

1. نصب پیش‌نیازها:
-----------------------
در ترمینال وارد شوید و اجرا کنید:
pip install -r requirements.txt

2. وارد کردن توکن ربات:
-----------------------
در فایل bot.py به جای YOUR_BOT_TOKEN_HERE توکن دریافتی از BotFather را وارد کنید.

3. اجرای ربات:
-----------------------
python bot.py

4. تست ربات:
-----------------------
در تلگرام به ربات بروید، دستور /start را بفرستید و سپس عکس واضح چهره خود را ارسال کنید.
برای فعال کردن حالت VIP (به صورت تست دستی) دستور /vip را بزنید.
